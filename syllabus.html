The programming projects are posted on GitHub. ####GitHub GitHub is a version control system that is widely used in the computing industry. A version control system is used to track versions of software as the software is updated. When used properly GitHub can be used to keep track of versions of your program and all updates, acting like a backup system and a failsafe system. If something goes wrong with an update you can easily revert to a previous version. For the small programming projects in this course we will likely not need that functionality, but programmers should be aware of systems like GitHub. ####LEARNING OUTCOMES: 1.
